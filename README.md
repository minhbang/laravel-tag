# Laravel Authorization
Quản lý Role - Permission của User

## Install

* **Thêm vào file composer.json của app**
```json
	"repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/minhbang/laravel-authority"
        }
    ],
    "require": {
        "minhbang/laravel-authority": "dev-master"
    }
```
``` bash
$ composer update
```
